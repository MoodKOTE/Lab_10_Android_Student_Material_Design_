# Student Material Design

**Student Material Design** — это учебное Android-приложение, разработанное в рамках лабораторных работ №10-11. Оно представляет собой интерактивный список студентов с возможностью просмотра подробной информации о каждом. Основная цель проекта — демонстрация ключевых принципов Material Design на практике с использованием современного подхода к созданию интерфейсов Jetpack Compose.

## Цель работы

Изучить и применить на практике принципы Material Design в Android-приложениях с использованием декларативного фреймворка Jetpack Compose. В ходе выполнения работы были освоены механизмы кастомизации цветовых схем, типографики, форм компонентов, а также добавления простых анимаций для создания современного и отзывчивого пользовательского интерфейса.

## Функциональность

Приложение обладает следующим набором функций:

-   **Отображение списка студентов:** Главный экран приложения содержит прокручиваемый список (LazyColumn), в котором отображается краткая информация о каждом студенте: фото, имя и возраст.
-   **Детальная информация:** При нажатии на кнопку со стрелкой справа от карточки студента, плавно раскрывается дополнительный блок с описанием (юмористической характеристикой) студента. Повторное нажатие скрывает это описание.
-   **Адаптивный дизайн (Material Theming):**
    -   **Цвет:** Приложение использует кастомную цветовую схему с основным зеленым цветом, определенную для светлой темы.
    -   **Форма:** Для аватарок студентов применена круглая форма (small), а карточки списка имеют динамичную форму со скругленными левым нижним и правым верхним углами (medium).
    -   **Типографика:** В проекте используются пользовательские шрифты (Montserrat и Abril Fatface) из Google Fonts, настроенные в соответствии с типаграфической шкалой Material Design.
-   **Поддержка тем:** Приложение полностью поддерживает светлую (Light Theme) и тёмную (Dark Theme) темы, которые переключаются автоматически в соответствии с системными настройками устройства.
-   **Анимация:** Раскрытие и скрытие блока с описанием студента сопровождается плавной анимацией изменения размера карточки.

## Технологии и инструменты

В проекте были использованы следующие технологии и библиотеки:

-   **Язык программирования:** Kotlin.
-   **UI-фреймворк:** Jetpack Compose.
-   **Библиотеки Compose:**
    -   `material3`: Для реализации компонентов Material Design 3 (Card, TopAppBar, Scaffold, иконки и т.д.).
    -   `material-icons-extended`: Для использования дополнительных иконок (ExpandMore, ExpandLess).
    -   `ui-tooling-preview`: Для предварительного просмотра UI-компонентов прямо в среде разработки.
-   **Управление зависимостями:** Gradle Kotlin DSL.
-   **Система контроля версий:** Git.
-   **Среда разработки:** Android Studio.
